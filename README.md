# Roblox Download Interface

Giao diện tải Roblox với đầy đủ tính năng: mô tả, ghi chú, hướng dẫn và thông tin liên hệ.

## Tính năng

- ✅ Giao diện đẹp mắt, hiện đại
- ✅ Nút tải Roblox
- ✅ Mô tả chi tiết về game
- ✅ Ghi chú quan trọng
- ✅ Hướng dẫn cài đặt từng bước
- ✅ Thông tin liên hệ đầy đủ
- ✅ Form liên hệ
- ✅ Responsive design (tương thích mobile)

## Cách sử dụng

1. Mở file `index.html` trong trình duyệt web
2. Hoặc sử dụng local server:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```
3. Truy cập `http://localhost:8000`

## Cấu trúc file

- `index.html` - File HTML chính
- `style.css` - File CSS cho styling
- `script.js` - File JavaScript cho tương tác
- `README.md` - File hướng dẫn này

## Tùy chỉnh

Bạn có thể tùy chỉnh:
- Link tải trong `script.js` (dòng 4)
- Thông tin liên hệ trong `index.html`
- Màu sắc trong `style.css` (tìm `#667eea` và `#764ba2`)

## Lưu ý

- Link tải mặc định là link chính thức của Roblox
- Form liên hệ hiện tại chỉ hiển thị thông báo, cần tích hợp backend để gửi email thực tế
